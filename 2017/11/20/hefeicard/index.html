扫码支付总账查询

REQ

参数 说明
pay_type 支付渠道
bill_date_start 账单日期
bill_date_end 账单日期

RESP

参数 说明
bill_date 账单日期
pay_type_name 支付渠道
check_result_code_name 对账结果
check_date 对账时间

扫码支付错账明细查询

REQ

参数 说明
total_id 结账id 总账索引

RESP

参数 说明
bill_date 账单日期
trade_no 订单号
pay_type 支付渠道
error_code_name 差错类型
error_amount 差错金额
order_status 订单状态
card_no 卡号
merchant_id 商户号
check_date 对账时间

差错明细平账

REQ

参数 说明
Id 结账id
set_flag 1:账平，0-账不平
remarks 备注

RESP

参数 说明

现金结账报表

REQ

参数 说明
check_date_start 结账日期开始
check_date_end 结账日期结束
deviceNo 设备号

RESP

参数 说明
deviceNo 设备号
check_date 结账时间
success_times 成功笔数
success_money 成功金额
error_times 失败笔数
error_money 失败金额
notsure_times 不确定次数
notsure_money 不确定金额
total_times 总笔数
total_money 总金额

交易明细报表

REQ

参数 说明
transdate_start 交易日期开始
transdate_end 交易日期结束
eccpan 卡号
cardtype 卡类型：20-PBOC卡，90-CPU卡
transtype 业务类型：cz-充值，sk-售卡，kcq-快充券
reserved 支付渠道
txnstatus 订单状态

RESP

参数 说明
txnseq 流水号
eccpan 卡号
cardtype 卡类型
transtype 业务类型
transdate 交易日期
transtime 交易时间
reserved_name 支付渠道
txnamt 交易金额
txnstatus 状态
txnstatus_name 状态
merchId 商户号
shopid 门店
deviceNo 终端

总账报表

REQ

参数 说明
transdate_start 交易日期开始
transdate_end 交易日期结束
eccpan 卡号

transtype 业务类型：cz-充值，sk-售卡，kcq-快充券
reserved 支付渠道
txnstatus 订单状态

RESP

参数 说明
cardtype 卡类型
transtype 业务类型
reserved_name 支付渠道
txnstatus_name 状态
sumtimes 笔数
sumamt 金额

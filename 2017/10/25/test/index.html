发卡记录查询

REQ

参数 类型 说明
fDateStart char 交易日期
fDateEnd char 交易日期
fDeptid char 上级机构代码链表，包括本身，以方便查询统计。格式为：省行,市行,支行,网点，中间以逗号’,’分隔
fResult char 交易结果交易结果，’0’ :交易成功 ‘1’:交易失败
fDevid char 设备编号
fOperator char 操作员
fAcno char 卡号/账号

fIdno char18 身份证号
fUsrno char20 用户号
fTram double 交易金额
fFee double 手续费
fType char10 开卡方式：银行卡、二代证
fDate1 char6 交易日期
fPay char6 支付方式：现金、转账
fDevls char20 设备交易流水
fAcctno char20 结账号

fZhname char40 总行名称

设备正常运行率统计（单机）

REQ

参数 类型 说明
fDateStart char 交易日期
fDateEnd char 交易日期
fDeptid char 上级机构代码链表，包括本身，以方便查询统计。格式为：省行,市行,支行,网点，中间以逗号’,’分隔
fMoneyDev char 是否为现金类设备 0 为否 1为是

RESP

参数 类型 说明
fTkj4 char kj4 系统维护时间
fPm char KJL字段的百分数表示
fType char 设备类型 (ATM/BST/DCT)
fSpc char 设备型号
fMark char 设备品牌，如“NCR”、“DIEBOLD”等
fInststyle char 安装方式：‘1’=大堂，‘2’=穿墙，‘3’=壁挂
fDevid char 设备编号
fRjkj char 平均每天（正常）运行的时间
fSjkj char kj（0,1,7,8,9） 总开机运行持续时间（分钟）
KJL char fSjkj/fTkjall 开机运行率
fTkjall char 设备应该总的运行开机时间
fTkj6 char 总通信故障时间
fTkj5 char 总暂停服务时间
fWdname char40 网点名称
fZhname char40 总行名称
fFdname char40 分行名称
fZhid char40 总行编号id
fWdid char40 网点编号id

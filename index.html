字段 别名 说明
F_ID fId 机构代码

F_LEVEL fLevel 级别： ‘1’=网点， ‘2’=支行， ‘3’=市行， ‘4’=省行
F_ADDR fAddr 所在地址
F_AREAID fAreaid 地区代码
F_UNM fUnm 属性： ‘1’=在行， ‘2’=离行， ‘3’=自助
F_DEVNUM fDevnum 设备数量（ 市行有效）。
F_MAPID fMapid 地图编号， 0=没有地图
F_MAPX fMapx 此机构在地图上的横坐标
F_MAPY fMapy 此机构在地图上的纵坐标
F_TEL fTel 联系电话
F_RECVER fRecver 事件接收操作员号（ 营业时段内）
F_RECVER1 fRecver1 事件接收操作员号（ 营业时段外）
F_VALID fValid 是否有效（ 机构记录被删除后置为无效）， 1－有效（ 默认）， 0－无效

F_SPC fSpc char20 设备型号， 取自 t_defdev 表

F_PC fPc char8 操作系统（ 见综合代码表(t_codes)OS）
F_BOX fBox int 存款钞箱个数

F_MONEYDEV fMoneydev char1 是否为现金类设备： ‘0’=否， ‘1’=是
F_FACID fFacid char8 设备厂商代码， 由程序根据设备型号设备型号表(t_defdev)得到
F_ISPID fIspid char8 设备维护商代码， 见设备服务商表(t_devisp)
F_DTOPEN fDtopen char8 投产日期
F_DTBX1 fDtbx1 char8 起保日期
F_DTBX2 fDtbx2 char8 终保日期
F_WORK1 fWork1 char4 起始营业时间， hhmm 格式
F_WORK2 fWork2 char4 结束营业时间， 同上
F_WORKDX fWorkdx double 每日营业时间， 本域由程序根据上两个字段自动计算
F_SEARCH fSearch char80 定位信息
F_CURVER fCurver char20 当前软件版本号
F_CODE fCode char20 设备序列号
F_LASTTX fLasttx char14 最后收到报文的时间

Maven

Apache Maven：是一个软件项目管理和构建工具。基于项目对象模型（POM）的概念，Maven可以通过中央信息库管理项目的构建，报告和文档。

  • 下载 ：
    • https://maven.apache.org/download.cgi
    • apache-maven-3.3.9-bin.zip并解压缩；
  • 安装 ：
    • 确保JAVA_HOME为jdk1.7；
    • apache-maven-3.3.9/bin的绝对路径添加到path环境变量；
    • cmd窗口中执行mvn -v，如果显示maven版本，表示安装成功；

  • IDE集成Maven ：

    • NetBeans： 从 6.7开始完全支持Maven, 从7.0+开始支持Maven 3；

    • Eclipse：

      • Help->Install new software->work with中填入
        http://download.eclipse.org/technology/m2e/releases 并回车；如下图：

      • 查询到“Maven Integration for Eclipse”后☑️

      • 页面最后一行“Contact all update sites during install to find require software”不选，加快安装速度；
      • next->accept license->finish，提示重启Eclipse->OK
      • Preference->Maven->installation,添加Maven_Home的路径，并起个名称，如下图：

      • M2Eclipse插件主页是http://www.eclipse.org/m2e/index.html ，可以查询相对应的版本号；

  • Maven主要功能：
    • 帮助开发者管理jar包，将开发所需jar包下载到本地
      • Maven Center：https://mvnrepository.com/
      • 本地 Center：~/.m2/repository/
    • 开发者导入基础项目（src+pom.xml）后，仅需关注dependency，添加到pom.xml->dependencies即可
